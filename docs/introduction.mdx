# Radix Agent Kit

An AI-first TypeScript toolkit for building intelligent agents that interact with the Radix blockchain through natural language.

## What is Radix Agent Kit?

**Radix Agent Kit** lets you build AI agents that talk to the Radix blockchain in plain English. Instead of writing complex blockchain code, just tell your agent what to do.

## âš¡ Quick Demo

```typescript
import { RadixAgent, RadixNetwork } from "radix-agent-kit";

const agent = new RadixAgent({
  networkId: RadixNetwork.Stokenet,
  openaiApiKey: process.env.OPENAI_API_KEY,
  mnemonic: process.env.RADIX_MNEMONIC
});

// Natural language blockchain interactions
await agent.run("What's my XRD balance?");
await agent.run("Send 10 XRD to Alice's account");
await agent.run("Create a token called GameCoin with 1M supply");
await agent.run("Stake 100 XRD with the highest APY validator");
```
