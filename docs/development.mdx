## Quick Setup

Get started with Radix Agent Kit in just a few steps:

### 1. Install the Package

```bash
npm install radix-agent-kit
```

### 2. Set Up Environment

Create a `.env` file:

```bash
OPENAI_API_KEY=your_openai_api_key_here
RADIX_MNEMONIC=your_24_word_mnemonic_phrase_here
```

### 3. Create Your First Agent

Create `index.js`:

```javascript
import { RadixAgent, RadixNetwork } from "radix-agent-kit";
import "dotenv/config";

const agent = new RadixAgent({
  networkId: RadixNetwork.Stokenet, // Use testnet for development
  mnemonic: process.env.RADIX_MNEMONIC,
  openaiApiKey: process.env.OPENAI_API_KEY,
  useMemory: true,
});

// Test the agent
async function main() {
  console.log("Agent created successfully!");
  console.log("Wallet:", agent.getWalletAddress());

  const response = await agent.run("What's my XRD balance?");
  console.log("Response:", response);
}

main().catch(console.error);
```

### 4. Run Your Agent

```bash
node index.js
```

## TypeScript Setup (Optional)

If you prefer TypeScript:

### 1. Install TypeScript Dependencies

```bash
npm install -D typescript @types/node tsx
```

### 2. Create `tsconfig.json`

```json
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "ESNext",
    "moduleResolution": "node",
    "esModuleInterop": true,
    "allowSyntheticDefaultImports": true,
    "strict": true,
    "skipLibCheck": true
  }
}
```

### 3. Create TypeScript Version

Rename `index.js` to `index.ts` and add types:

```typescript
import { RadixAgent, RadixNetwork } from "radix-agent-kit";
import "dotenv/config";

const agent = new RadixAgent({
  networkId: RadixNetwork.Stokenet,
  mnemonic: process.env.RADIX_MNEMONIC!,
  openaiApiKey: process.env.OPENAI_API_KEY!,
  useMemory: true,
});

async function main(): Promise<void> {
  console.log("Agent created successfully!");
  console.log("Wallet:", agent.getWalletAddress());

  const response = await agent.run("What's my XRD balance?");
  console.log("Response:", response);
}

main().catch(console.error);
```

### 4. Run with TypeScript

```bash
npx tsx index.ts
```

## Getting Your Keys

### OpenAI API Key

1. Go to [OpenAI Platform](https://platform.openai.com/api-keys)
2. Create a new API key
3. Copy it to your `.env` file

### Radix Mnemonic

- **For testing**: Generate a new wallet with the Radix Wallet app
- **For development**: Use the agent to generate one:

```javascript
import { RadixMnemonicWallet, RadixNetwork } from "radix-agent-kit";

const wallet = RadixMnemonicWallet.generateRandom({
  networkId: RadixNetwork.Stokenet,
});

console.log("Mnemonic:", wallet.getMnemonic());
console.log("Address:", wallet.getAddress());
```

<Warning>
  Never use real funds on testnet. Always use Stokenet for development and
  testing.
</Warning>

## Common Operations

### Check Balance

```javascript
const response = await agent.run("What's my balance?");
```

### Transfer Tokens

```javascript
const response = await agent.run("Send 10 XRD to account_tdx_...");
```

### Create Token

```javascript
const response = await agent.run(
  "Create a token called MyToken with 1000 supply"
);
```

### Stake XRD

```javascript
const response = await agent.run("Stake 100 XRD");
```

## Error Handling

Always wrap agent calls in try-catch:

```javascript
try {
  const response = await agent.run("Your command here");
  console.log(response);
} catch (error) {
  console.error("Error:", error.message);
}
```
