Your AI agent comes with **8 specialized tools** that handle all Radix blockchain operations. Each tool translates natural language into blockchain actions.

## ü§ñ How Tools Work

```typescript
// User says this:
await agent.run("Send 100 XRD to Alice");

// Agent automatically:
// 1. Understands intent
// 2. Selects TransferTokensTool
// 3. Executes blockchain transaction
// 4. Returns friendly response
```

<Info>
**Pro Tip**: You can also call tools directly without the AI agent for programmatic usage.
</Info>

## üß∞ All 8 Tools

### 1. üí∞ GetAccountInfoTool

**What it does**: Gets account details and metadata

```typescript
// Natural language
await agent.run("Show my account details");
await agent.run("Is this address valid: account_tdx_...");

// Direct usage
const tool = new GetAccountInfoTool(gateway, builder, wallet, networkId);
const result = await tool.call("account_tdx_...");
```

**Returns**: Account address, public key, metadata, validation status

---

### 2. üìä GetBalancesTool  

**What it does**: Checks token balances for any account

```typescript
// Natural language

await agent.run("Show all my token balances");
await agent.run("How much MyToken do I have?");

// Direct usage
```

**Returns**: XRD balance, all token balances, NFT counts, LSU balances

---

### 3. üí∏ TransferTokensTool

**What it does**: Transfers tokens between accounts


**Parameters**: Destination address, amount, token type (defaults to XRD)

## üõ†Ô∏è Direct Tool Usage

You can use tools directly without the AI agent:


## üéØ Tool Input Formats

Each tool expects specific input formats:

| Tool | Input Format | Example |
|------|-------------|---------|
| **GetAccountInfo** | `address` | `account_tdx_...` |
| **GetBalances** | `address` | `account_tdx_...` |
| **TransferTokens** | `address,amount,token` | `account_tdx_...,100,xrd` |
| **CreateFungible** | `name,symbol,supply,decimals` | `GameCoin,GAME,1000000,18` |
| **StakeXRD** | `action,validator,amount` | `stake,validator_tdx_...,100` |
| **AddLiquidity** | `pool,amount1,amount2` | `component_tdx_...,1000,500` |
| **SwapTokens** | `pool,from,to,amount` | `component_tdx_...,resource1,resource2,100` |
| **CallComponent** | `component,method,args...` | `component_tdx_...,get_price,arg1` |

## üöÄ Adding Custom Tools

Create your own tools for specific use cases:

```typescript
import { RadixTool } from "radix-agent-kit";

class MyCustomTool extends RadixTool {
  name = "my_custom_tool";
  description = "Does something specific for my app";

  async call(input: string): Promise<string> {
    // Your custom logic here
    const [param1, param2] = input.split(",");
    
    // Use Radix integration
    const result = await this.gateway.someCustomOperation(param1, param2);
    
    return `Custom operation completed: ${result}`;
  }
}

// Add to agent
const agent = new RadixAgent({
  // ... config
  customTools: [new MyCustomTool(gateway, builder, wallet, networkId)]
});
```
