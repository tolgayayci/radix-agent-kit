## ü§ñ AI Agent Features

### Natural Language Processing
Your agent understands plain English commands and translates them into blockchain actions.

```typescript
// Instead of complex blockchain code
await agent.run("Send 100 XRD to Alice");
await agent.run("Create a token called GameCoin");
await agent.run("Swap XRD for MyToken");
```

### Memory & Context
Agents remember conversation history and can reference previous actions.

```typescript
const agent = new RadixAgent({
  useMemory: true  // Remembers conversation
});

await agent.run("Create a token called MyToken");
await agent.run("Now send 100 MyToken to Alice");  // Remembers MyToken
```

## üîê Security Features

### Secure Key Management
- **AES-256-GCM encryption** for stored keys
- **PBKDF2 key derivation** with high iteration count
- **Environment variable** integration
- **Never logs** private keys or mnemonics


### Transaction Validation
- **Address format** validation
- **Balance checking** before transfers
- **Rate limiting** to prevent spam
- **Input sanitization** for all parameters

### Access Control
